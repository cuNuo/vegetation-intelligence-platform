# 20260623-1039 Agent设计技能检索与安装

## 前置说明

- 用户要求使用 `find-skills` 查找 agent 设计相关 skill，并把最切合的安装在项目内。
- 本轮按 `find-skills` 要求使用 Skills CLI 搜索，并优先检查来源与安装量。

## 搜索结果摘要

- `agent design`
  - `thedesignproject/agent-skills@agentic-design-systems`：49 installs
  - `fusengine/agents@agent-design`：34 installs
- `agent builder`
  - `starchild-ai-agent/official-skills@agent-builder`：23 installs
  - 其他候选安装量也偏低
- `multi agent`
  - `andrelandgraf/fullstackrecipes@ai-agent-workflow`：133 installs
  - `majesticlabs-dev/majestic-marketplace@multi-agent-architecture`：44 installs
- `langchain agent`
  - `langchain-ai/langchain-skills@ecosystem-primer`：873 installs，官方 LangChain 来源
  - 其他 LangChain agent/retrieval 候选安装量较低
- `rag agent`
  - `yonatangross/orchestkit@rag-retrieval`：709 installs
  - `qodex-ai/ai-agent-skills@rag-agent-builder`：183 installs

## 取舍

- 通用 “agent design/builder” 搜索结果安装量普遍较低，不适合直接作为项目主 skill。
- 当前项目已经使用 LangChain、RAG、工具调用、用户确认执行和 PostgreSQL 持久化，因此官方 LangChain `ecosystem-primer` 最适合作为外部参考技能。
- 为了贴合本项目，还新增了项目专用 `vegetation-agent-designer`，明确植被指数 Agent 的安全边界、RAG 顺序、自定义指数校验、PostgreSQL 持久化和验收检查。

## 安装结果

- 已通过 Skills CLI 项目级安装：
  - `.agents/skills/ecosystem-primer`
  - `skills/ecosystem-primer`
  - 以及 CLI 自动复制到多个 agent 兼容目录
- 已新增项目专用 skill：
  - `.agents/skills/vegetation-agent-designer/SKILL.md`
  - `.codex/skills/vegetation-agent-designer/SKILL.md`
- `npx skills list --json` 已确认项目级技能包含：
  - `ecosystem-primer`
  - `vegetation-agent-designer`

## 后续使用

- 做 LangChain API/生态选型时，优先使用 `ecosystem-primer`。
- 做本项目 Agent 行为、安全、工具、RAG、执行闭环改造时，优先使用 `vegetation-agent-designer`。
- 安装新 skill 后需要重启 Codex 才能被新会话自动识别。
